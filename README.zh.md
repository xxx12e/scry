# 🏮 Fathom

### Everything 找的是文件名。**Fathom 找的是意思。**

秒级、全盘的**语义**文件搜索，**100% 离线**运行。
按你**记得的意思**去搜——不用记准确的字眼——就能找到文件。你的文件从不离开这台电脑。

`全盘语义搜索 · 秒懂你要找什么 · 纯本地、不联网、隐私零外泄`

**Windows · 支持 CPU / GPU · 无云端、无账号、无遥测**

## 为什么用 Fathom

你记得**某个文件是讲什么的**。但你记不住它的名字、在五个盘的哪一个、当初写的是"发票"还是"invoice"还是"账单"。关键词搜索恰恰在这里失灵。

Fathom 按**意思**搜索。它骑在 Windows 已经建好的索引之上，加一层神经网络重排；当你要最好的质量时，它还能对你选定的文件夹建立**自己的深度语义索引**。全部在你自己的机器上，可断网运行，什么都不往外传。

> 这就是 **Everything** 本该进化成的样子：一样的"全盘、即时"手感——但它听得懂你。

## 一个搜索框，三种搜法

| 模式 | 干什么 | 需要设置 |
|---|---|---|
| 🌐 **全盘语义** | 按意思搜全机器每个文件的**内容**。骑 Windows 内容索引拿候选，再用神经网络按意思重排。 | **不用。** 打开即搜。 |
| ⚡ **文件名** | 像你熟悉的那样，全盘秒搜文件名。 | 不用。 |
| 🎯 **深度语义（掌控模式）** | 指定文件夹（或整个盘），它建**自己的**深度索引——质量最高，能命中**零字面重合**的改写。首次全量扫描，之后只碰有变化的文件，并在后台自动保持最新。 | 每个文件夹点一下。 |

此外：**中英双语界面（一键切换）**、全键盘操作（`回车`搜索、`↑/↓`移动、`回车`打开）、高亮片段直接告诉你**为什么**这个文件匹配，还有一个**免装 Python** 的便携版，在任意 Windows 电脑上双击即用。

## 隐私立场（这就是全部意义）

- **全程本地。** 嵌入模型已内置。拔掉网线，一切照常工作。
- **只监听 `127.0.0.1`。** 不对局域网开放，更不用说互联网。
- **无账号、无遥测、无"匿名使用统计"。** 根本没有服务器可以联。
- 你的索引和设置存在自己磁盘的 `~/.localsearch_index`。

## 快速上手

**从源码运行**

```bash
pip install -r requirements.txt
python -m localsearch.app          # 浏览器打开 http://127.0.0.1:8731
```

或直接双击 **`run.bat`**（Windows）。

**便携版（免装 Python）** — 打包成一个自带 CPU 模型的独立文件夹，双击 `.exe` 即可。任意 Windows 电脑、完全离线。直接下 [Releases](../../releases) 里的压缩包即可；打包方法见 [`localsearch/README.md`](localsearch/README.md)。

## 工作原理

```
        你的查询（"那个水库跌到历史最低的文件"）
                              │
        ┌─────────────────────┼───────────────────────────┐
     全盘                   文件名                      深度语义
   (Windows 内容索引)    (Everything 式，MFT 文件名)   (对你选的文件夹建的 FAISS 索引)
        │                                                    │
        └──────────────►  神经网络重排（bge 向量）  ◄──────────┘
                              │
                     排好序的文件 + 精确到命中的那一句，高亮显示
```

- **向量模型：** 现成的 `BAAI/bge-base-en-v1.5`（768 维）。**永不训练。**
- **检索：** 精确 FAISS（fp16 标量量化——内存/磁盘减半，仍是穷举精确）+ BM25 词法一臂融合；CJK 感知分块 + 二元组分词，中文文档能被完整搜到。
- **即时模式：** 通过 `Search.CollatorDSO` 直查系统内容索引，再嵌入重排——**零预索引**的全盘内容搜索。
- **稳如老狗：** 索引写入防掉电（fsync + 代号戳，撕裂的保存能被检测并隔离）、退出前自动保存、建索引期间进度条实时刷新且能同时搜索、单线程看护器扛得住批量拷贝。

诚实说清能力边界：Fathom 的语义质量是"真的好用"，不是魔法——小型本地模型的上限远低于数据中心。内置模型以英文为主；中文能搜（结构 + 词法 + 二元组 BM25 三管齐下），如果中文是你的主力语言，可以换上 `bge-base-zh`。它的目标不是在跑分上击败云端，而是给你**云端级的便利 + 零隐私代价**。

## 项目结构

```
localsearch/          引擎 + 本地网页界面（这就是产品本体）
  ├─ engine.py        总调度：扫描 → 解析 → 分块 → 嵌入 → 索引 → 搜索
  ├─ index.py         双索引：fp16 FAISS 稠密 + BM25，防掉电持久化
  ├─ winsearch.py     "即时全盘"的关键突破（Windows 内容索引）
  ├─ search.py        混合检索 + 最佳句片段
  ├─ watcher.py       防抖后台增量重索引
  ├─ app.py           Flask 后端（仅 127.0.0.1）+ JSON API
  └─ web/index.html   单页界面，双语，零构建步骤
demo_*.py             可直接运行的端到端示例
run.bat               双击启动
```
